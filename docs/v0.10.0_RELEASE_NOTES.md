# xAI SDK Go v0.10.0 Release Notes

**Release Date:** December 10, 2025

## üéØ Focus: Python SDK v1.5.0 Parity

This release brings the Go SDK up to parity with Python SDK v1.5.0 by adding support for verbose streaming, attachment search, enhanced inline citations, and new collections search options.

## ‚ú® New Features

### Verbose Streaming Include Option

A new include option for more detailed streaming information during agentic tool calls:

```go
req := chat.NewRequest("grok-3",
    chat.WithMessage(chat.User(chat.Text("Search and analyze..."))),
    chat.WithServerTool(chat.WebSearchTool()),
    chat.WithInclude(chat.IncludeVerboseStreaming),
)
```

### Enhanced Inline Citations

Inline citations now include an `EndIndex()` method for precise citation positioning:

```go
for _, citation := range resp.InlineCitations() {
    fmt.Printf("Citation %s: characters %d-%d\n",
        citation.ID(),
        citation.StartIndex(),
        citation.EndIndex(),  // NEW in v0.10.0
    )
}
```

### Collections Search Enhancements

New options for customizing collections search behavior:

```go
// With custom instructions
tool := chat.CollectionsSearchTool([]string{"col-1", "col-2"},
    chat.WithCollectionsLimit(10),
    chat.WithCollectionsInstructions("Focus on recent API documentation"),
)

// With specific retrieval modes
tool := chat.CollectionsSearchTool([]string{"col-1"},
    chat.WithSemanticRetrieval(),  // or WithHybridRetrieval(), WithKeywordRetrieval()
)
```

### Attachment Search Tool

The document search tool has been renamed to attachment search for clarity:

```go
// Search within uploaded file attachments
tool := chat.AttachmentSearchTool(
    chat.WithAttachmentLimit(5),
)

req := chat.NewRequest("grok-3",
    chat.WithMessage(chat.User(chat.Text("Search my attachments for API keys"))),
    chat.WithServerTool(tool),
)
```

### GetToolCallType Utility Function

New utility function to retrieve tool call types:

```go
for _, tc := range resp.ToolCalls() {
    typeStr := chat.GetToolCallType(tc)
    fmt.Printf("Tool call type: %s\n", typeStr)
    // Returns: "client_side", "web_search", "attachment_search", etc.
}
```

## ‚ö†Ô∏è Breaking Changes

### Renamed APIs

The following APIs have been renamed to better reflect their purpose:

| Old Name | New Name |
|----------|----------|
| `DocumentSearchTool()` | `AttachmentSearchTool()` |
| `WithDocumentLimit()` | `WithAttachmentLimit()` |
| `IncludeDocumentSearchCallOutput` | `IncludeAttachmentSearchCallOutput` |
| `ToolCallTypeDocumentSearch` | `ToolCallTypeAttachmentSearch` |

### Migration Guide

Update your code as follows:

```go
// Before (v0.9.0)
tool := chat.DocumentSearchTool(chat.WithDocumentLimit(5))
req.SetInclude(chat.IncludeDocumentSearchCallOutput)
if tc.Type() == chat.ToolCallTypeDocumentSearch { ... }

// After (v0.10.0)
tool := chat.AttachmentSearchTool(chat.WithAttachmentLimit(5))
req.SetInclude(chat.IncludeAttachmentSearchCallOutput)
if tc.Type() == chat.ToolCallTypeAttachmentSearch { ... }
```
