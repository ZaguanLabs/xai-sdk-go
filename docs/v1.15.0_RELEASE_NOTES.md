# xAI SDK Go v1.15.0 Release Notes

**Release Date:** June 4, 2026

## 🎯 Focus: Python SDK v1.15.0 Parity

This release ports the public API additions from the official Python SDK through v1.15.0, including the upstream v1.13.0 batch video extension helper and v1.14.0 image search option for web search tools.

## ✨ Added

### Protobuf and Service Surface Parity
- Added `WebSearch.enable_image_search` to match Python `tools.web_search(enable_image_search=...)`.
- Added `BatchRequest.video_extension_request` to support batch video extension requests.

### Python-Style Client Conveniences
- Added `video.PrepareExtension()` and `(*video.Client).PrepareExtension()`.
- Added `batch.RequestFromVideoExtensionRequest()` and `batch.PrepareVideoExtensionRequest()`.
- Extended `batch.Client.Add()` to accept `*xaiv1.ExtendVideoRequest`.
- Added `tools.WebSearchOptions.EnableImageSearch`.

## 🐛 Fixed

- Kept `google/rpc/status.proto` mapped to the canonical `google.golang.org/genproto/googleapis/rpc/status` package to avoid duplicate protobuf descriptor registration.
